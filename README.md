## Pré-requisitos

- git (colocar o link do git)
- node (link)

## Versões

- Cypress xx.x.
- Cucumber


## Instalação

Baixar o repositório:

```bash
git clone https://github.com/gustavo-pereira-sofist/projeto-sofist
```

Instalar o pacote:

```bash
npm i
```

## Execução

```bash

# Executa no modo browser
npx cypress open

# Executa no modo comando
npx cypress run
```
